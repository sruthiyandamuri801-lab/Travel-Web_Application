# Travel-Web_Application
Project Description

The Travel Web Application is a responsive and user-friendly website developed using HTML and CSS. It is designed to help users explore popular travel destinations, view tour packages, and contact the travel agency easily. The website focuses on clean design, simple navigation, and modern layout techniques to enhance user experience.

This project demonstrates front-end web development skills, including responsive design, layout structuring, and visual styling.
